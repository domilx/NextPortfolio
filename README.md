

### Quick Preview - https://domidev.net/

### Prerequisites

* [NodeJS](https://nodejs.org/en/)

### Execution

1. Clone the directory.
```bash
$ git clone https://github.com/domilx/ReactPortfolio.git
```
2. Install the prerequisites above.
```bash
$ npm install
```
3. Run the project.
```bash
$ npm run dev
```
